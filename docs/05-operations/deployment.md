# 배포

## 한 줄 요약
개발한 앱을 실제 서비스 환경에 올리는 방법을 정리하는 문서입니다.

## 쉽게 설명하면
- 내 컴퓨터에서만 실행되는 상태를, 실제 사용자도 접속 가능한 상태로 바꾸는 과정입니다.

## 배포 대상
- Azure (상용 환경 기준)

## 현재 결정사항
- Azure를 기준으로 배포 전략을 작성한다.
- 초보자 친화적인 단계형 배포를 사용한다.
- 1단계: 웹 화면부터 먼저 배포한다.
- 2단계: 백엔드 API를 연결한다.
- 3단계: 데이터베이스와 모니터링을 확장한다.
