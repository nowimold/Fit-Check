# 기능 요구사항

## 한 줄 요약
앱이 실제로 해야 하는 기능 목록입니다.

## 쉽게 설명하면
- 버튼을 누르면 어떤 일이 일어나야 하는지 적습니다.
- 사용자 입장에서 "할 수 있어야 하는 일"을 적습니다.

## 현재 결정사항
- 사용자는 할 일 체크리스트를 작성/완료할 수 있어야 한다.
- 사용자는 캘린더로 일별 실천 기록을 확인할 수 있어야 한다.
- 앱은 사용자를 독려하는 피드백을 제공해야 한다.

## 다음에 사용자에게 물어볼 질문
- 독려 기능은 어떤 트리거에서 보여줄까요? (예: 3일 연속 성공, 오늘 미실행)
